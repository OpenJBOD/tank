Legend

Hostname - Sets the hostname the device will use for DHCP requests.

IP Method - Decides whether to use DHCP or static IP addressing.

IP Address - IPv4 Address of the device, only used with the static option above.

Subnet Mask - IPv4 subnet mask of the device, only used with the static option above.

Gateway - IPv4 gateway of the device, only used with the static option above.

DNS - IPv4 DNS IP for the device, only used with the static option above.

IPv6 Configuration - Select SLAAC, DHCPv6, Static, or Disabled.

IPv6 Address & Gateway - Required when Static IPv6 mode is selected.

IPv6 DNS - Optional IPv6 resolver when using Static mode.
