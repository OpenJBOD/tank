About OpenJBOD

Project Overview

OpenJBOD Tank is a software product of the OpenJBOD Project (GitHub)

Tank is free and open source software licensed under the Apache 2.0 license. The source code for Tank can be found on GitHub at https://github.com/openjbod/tank.

The OpenJBOD project would not be possible without the help of its contributors: dviske (GitHub), and Natan Keddem (GitHub).

The OpenJBOD Controller hardware is open source and licensed under the CERN OHL v2 - Permissive license.
