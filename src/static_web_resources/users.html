User Management

Current Users - List of all configured users on the system. Maximum of 5 users are supported.

Add New User - Create a new user account with username and password. Usernames must be unique and are case-sensitive.

Change Password - Update the password for an existing user account.

Delete User - Remove a user account from the system. This action cannot be undone.

Limits

Users - Up to 5 accounts.

Username - 1 to 31 characters, unique and case-sensitive. Longer names are rejected (not truncated).

Password - 8 to 64 characters. Stored only as a salted SHA-256 hash, never in plain text. These limits are enforced by the device, so the values you set here will always work at login.
