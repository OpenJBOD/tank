Settings - Console

Legend

Both consoles run the same interactive shell and log output.

Changes take effect on the next reboot, not immediately.

UART console is the primary recovery console. Disabling it leaves only the USB console (and the network) to manage the device — do not disable both.

USB console enumerates this device as a USB serial port ("OpenJBOD Tank Console") when connected to a host.
