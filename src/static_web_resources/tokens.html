API tokens authenticate programmatic requests without a username and
                           password. Send one as an HTTP header:
                           Authorization: Bearer <token>.

                        
The full token is shown once at creation; only a hash is stored on
                           the device, so a lost token cannot be recovered — revoke it and
                           create a new one.

                        
Tokens persist across reboots and remain valid until revoked here.
