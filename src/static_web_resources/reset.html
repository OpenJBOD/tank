Settings - Reset

Device Reboot

Immediately restarts the RP2040 controller without touching stored configuration.

Factory Reset

Erases all persisted settings, restores defaults, and then restarts the controller.

Backup Configuration

Download the current configuration (settings.dat) to your computer so you can restore it later or copy it to another board.

Restore Configuration

Upload a previously downloaded settings.dat. The device overwrites its configuration with the file and then reboots to apply it.
