Legend

HTTP Server - Enable or disable the HTTP web interface on port 80 (or custom port). This provides unencrypted access to the device.

HTTP Port - Port number for the HTTP server. Default is 80.

HTTPS Server - Enable or disable the HTTPS (encrypted) web interface on port 8443 (or custom port). This provides encrypted access to the device.

HTTPS Port - Port number for the HTTPS server. Default is 443.

Custom Certificates - Use uploaded certificates instead of the default self-signed certificate. When enabled, the server will use uploaded certificates for HTTPS connections.

Certificate Management - Upload new certificates that can be used with the HTTPS server. Certificates are stored separately from the setting to use them.
