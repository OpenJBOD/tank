Enable External Fan Control - This option disables the automatic fan curve response and lets you set the fan speed manually.

Update Interval - How often (in milliseconds) the fan control system checks the temperature and adjusts fan speed. Lower values provide more responsive control but use more CPU.

Hysteresis - Temperature difference (as percentage of temperature span) required before changing fan speed. Helps prevent oscillation between fan speeds.

Fan Curve - Five temperature/fan speed points that define the fan response curve. The system uses linear interpolation between points to determine fan speed at any temperature.

Temperature Range - Valid range is -40°C to +125°C (DS18B20 sensor limits).

Fan Speed Range - 0% to 100% where 0% is off and 100% is maximum speed.
